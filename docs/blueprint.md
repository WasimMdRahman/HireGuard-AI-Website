# **App Name**: ResumeRisk AI

## Core Features:

- Resume Scoring: Scores resumes based on alignment with the job description, providing a percentage match.
- Risk Flagging: Flags potential risks and inconsistencies within the resume based on the job description using a generative AI tool.
- Interview Question Generation: Generates targeted interview questions based on identified risks and skill gaps.
- Resume Upload: Allows users to upload candidate resumes in common formats (PDF, DOCX).
- Job Description Input: Allows users to input job descriptions via text pasting.
- Demo Video Embedding: Embeds a demo video showcasing the product's core functionality.
- Private API Key Integration: Enables users to integrate their own AI API key for private and secure use.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to convey trust and professionalism. It is a versatile color that will stand out against a light background.
- Background color: Off-white (#F9FAFB), creating a clean and minimal feel. This is subtly of the same hue as the primary color.
- Accent color: Soft periwinkle (#93C5FD) for interactive elements, providing a subtle contrast.
- Body and headline font: 'Inter', a sans-serif font that provides a neutral, clean, enterprise-grade feel.
- Clean and minimal layout with clear sectioning and generous use of white space.
- Subtle, professional animations such as smooth scrolling and slight hover effects on CTAs.
- Use simple, professional icons to represent key features and benefits.
